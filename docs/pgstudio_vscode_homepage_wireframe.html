SELECT
  date_trunc('day', created_at) AS day,
  COUNT(*) AS orders,
  SUM(amount) AS revenue
FROM orders
WHERE created_at >= NOW() - INTERVAL '7 days'
GROUP BY day ORDER BY day;

🚀 Quick Start

Install

1
Open VS Code → press Ctrl+Shift+X to open Extensions
2
Search for PostgreSQL Explorer and click Install
3
Or install via terminal:
code --install-extension ric-v.postgres-explorer

First Connection

4
Click the 🐘 elephant icon in the Activity Bar
5
Click Add Connection → fill in your host, database, credentials
6
Click connect and start exploring 🎉
