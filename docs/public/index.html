🚀 Install and Start in 2 Minutes

PgStudio is free and optimized for day-to-day PostgreSQL work inside VS Code.

  1. Open Extensions in VS Code (Ctrl+Shift+X).
  2. Search for ric-v.postgres-explorer.
  3. Install the extension and create your first connection.
  4. Open a .pgsql notebook and run your first query.
code --install-extension ric-v.postgres-explorer

Then open the PgStudio icon in the Activity Bar and run your first notebook cell.
